# Business Context

## Overview

ModelTools is an open-source MCP (Model Context Protocol) server that provides AI assistants with a comprehensive toolkit for document processing, code analysis, and system operations.

## Target Users

- AI coding assistants (Claude Code, etc.)
- Developers using AI tools that support MCP
- Teams building AI-powered workflows

## Value Proposition

- Universal compatibility with any MCP-supporting model
- Complete document lifecycle (read, write, convert, analyze)
- File system and code repository operations
- No vendor lock-in — open source and self-hostable

## Distribution

- **npm**: @adelvillar/modeltools
